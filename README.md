## 프로젝트 소개

### 프로젝트 개요/동기

컴퓨터공학과 졸업작품 제작할 때 직접 게임서버를 만들었었으나,  
프로토콜을 효율적으로 정립하지 않고 진행했던 경험이 있음.  
그래서 이번에는 Protobuf를 통해 패킷을 효율적으로 만들고 서버를 올려서  
상용 서버처럼 사용할 수 있게끔 만들면서 공부하고자 함.
